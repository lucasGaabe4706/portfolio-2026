Antes de entrar...

Recomendo dar uma lida no Sobre primeiro — vai te ajudar a entender melhor o que o HubOS faz. Mas fique à vontade para pular direto para a demonstração se preferir.

O HubOS é uma versão demonstrativa — com dados mockados e produtos fictícios — de uma plataforma real que desenvolvi para a empresa AR70.

O sistema original conta com integração ao ERP SAP Business One, múltiplos usuários, controle de autenticação e diversas outras funcionalidades que não puderam ser disponibilizadas por envolverem dados internos da empresa.

Esta demo existe para mostrar a interface e o funcionamento do sistema de forma fiel, sem expor nenhuma informação real.
